# Ecommerce-Project-SQL-Pandas-
This project focuses on analyzing an E-Commerce dataset using Python, MySQL, Pandas, NumPy, Matplotlib, and Seaborn. The objective is to extract meaningful business insights from customer, order, payment, product, and seller data through SQL queries and data visualization techniques.
The analysis helps understand customer behavior, sales performance, revenue trends, retention rates, and product category performance, enabling data-driven business decisions.

# 🛠️ Technologies Used
Python
MySQL
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook
# 📂 Dataset Components

The project utilizes multiple interconnected e-commerce tables:

Customers
Orders
Products
Payments
Order Items
Sellers

These tables are integrated using SQL joins to perform advanced business analysis.

# 📊 Business Questions Solved
## Basic Analysis
List all unique cities where customers are located.
Count the number of orders placed in 2017.
Calculate total sales by product category.
Determine the percentage of orders paid in installments.
Count customers from each state.
## Intermediate Analysis
Analyze customer distribution across product categories.
Calculate monthly order trends for 2018.
Find the average number of products per order by customer city.
Calculate revenue contribution percentage for each product category.
Identify the relationship between product price and purchase frequency.
## Advanced Analysis
Rank sellers based on total revenue generated.
Calculate moving averages of customer order values.
Generate cumulative monthly sales trends.
Measure Year-over-Year (YoY) sales growth.
Calculate customer retention rates.
Identify the top 3 highest-spending customers for each year.
# 📈 Key Insights Generated
Customer geographical distribution analysis.
Product category revenue contribution.
Monthly and yearly sales performance tracking.
Seller revenue ranking and performance evaluation.
Customer purchasing behavior analysis.
Revenue growth and retention metrics.
Product pricing impact on sales volume.
